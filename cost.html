Methodology: Investment Payback = Build Cost ÷ Monthly Value, using the midpoint of each use case's stated annual value range. Because these use cases operate against an already-large loan book / transaction volume / delinquent-account pool, even a modest build cost pays back in weeks, not months — the real differentiator between scenarios is total build cost and time-to-production, not payback speed (both scenarios pay back inside a month).
