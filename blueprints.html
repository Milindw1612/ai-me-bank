Use Case #1

MSME Credit CAM Prep

Problem: MSME loan file assembly (GST returns, AA bank feeds, ITRs, CIBIL, financials) takes 5–7 days per application against a ~₹1.02 lakh crore MSME book (~20% of lending portfolio), capping origination velocity.

Agent Architecture

  • Financial Ingestion Agent — OCR: ITRs, P&L, balance sheets
  • Bureau & Tax Verification Agent — CIBIL, AA feeds, GSTR-1/3B (runs in parallel with Financial Ingestion)
  • Risk & Cash-Flow Agent — joins both, flags anomalies
  • CAM Drafting Agent — recommended pricing, pre-filled memo
  • Human Credit Officer — final sign-off, always

Tech Stack

LangGraphClaude Sonnet 4.5Claude Haiku 4.5Account Aggregator API CIBIL APIGST APIOCR ServiceAWS RDS + pgvectorLangfuse

Budget & ROI

Turnaround time drops from 5–7 days to under 4 hours. Origination capacity increases 25–35% with no added headcount. ₹45–60 Cr/yr processing-cost saving, payback in 3–4 months.

Risk Register

Low–Medium Read-only AA/bureau integration; final lending authority stays human. Primary risk is data-quality drift in OCR extraction — mitigated by a confidence-threshold fallback to manual review.

Delivery Roadmap

Wk 1–3Platform setup, AA/bureau API contracts signed
Wk 4–7Financial Ingestion + Bureau/Tax agents built
Wk 8–10Risk & Cash-Flow + CAM Drafting agents built
Wk 11–12UAT with credit officers, pilot cohort
Wk 13–14Go-live, hypercare
Use Case #2

Fraud Detection & Investigation

Problem: Fraud teams triage thousands of daily alerts across cards/UPI/IMPS; most are false positives. Manual investigation (device, IP, behavior, merchant checks) takes 20–40 minutes per case, delaying genuine fraud response.

Agent Architecture

  • Device/IP & Network Agent — device fingerprint, IP reputation
  • Behavior Pattern Agent — spend/velocity anomaly detection (parallel with Device/IP)
  • Merchant/Beneficiary & AML Agent — merchant risk, AML watchlists (parallel)
  • Recommendation Agent — joins all 3, scores fraud probability
  • Fraud Analyst — reviews only escalated cases (0.3–0.7 probability band)

Tech Stack

LangGraphClaude Haiku 4.5Device/IP Intelligence APIAML Watchlist API Core Banking APIAWS RDSLangfuse

Budget & ROI

Investigation time drops from 20–40 minutes to under 1 minute. False positives cut 30–40%. Fraud loss rate falls from ~0.05–0.10% to 0.02–0.04% — ₹100–150 Cr/yr avoided losses, payback under 3 months.

Risk Register

Low–Medium Read-only monitoring layer; auto-freeze thresholds need governance sign-off before going live — false-freeze rate is tracked as a customer-experience KPI, not just a fraud KPI.

Delivery Roadmap

Wk 1–2Signal-provider API contracts (device/IP, AML)
Wk 3–53 parallel signal agents built
Wk 6–7Recommendation agent + auto-action thresholds
Wk 8–9Shadow-mode run alongside existing rules engine
Wk 10Go-live, analyst hypercare
Use Case #3

Collections & Delinquency Recovery

Problem: 1–90 DPD retail/card accounts number in lakhs monthly; human collections teams reach only 20–30% via manual calls/generic SMS, letting recoverable accounts roll into higher NPA buckets.

Agent Architecture

  • Risk/Behavior Agent — payment history scoring
  • Channel Agent — SMS/WhatsApp/voice selection
  • Negotiation Agent — settlement/EMI within pre-approved policy bands
  • Escalation Agent — flags complex/aggressive cases with full context
  • Human Collections Officer — handles escalations only

Tech Stack

LangGraphClaude Haiku 4.5WhatsApp APISMS Gateway IVRBureau Payment History APILangfuse

Budget & ROI

Portfolio coverage jumps from 20–30% to 100% daily. Recovery uplift of 15–25% in the 1–30 DPD bucket. Very High ROI, NPA-linked and board-tracked, payback under 3 months.

Risk Register

Low Advisory + negotiation strictly within pre-set policy bands; human sign-off required on every exception. Tone/compliance of automated outreach is monitored against RBI fair-practice collection norms.

Delivery Roadmap

Wk 1–2Channel provider integration (WhatsApp/SMS/IVR)
Wk 3–5Risk/Behavior + Channel agents built
Wk 6–8Negotiation + Escalation agents, policy bands encoded
Wk 9Pilot on one DPD bucket
Wk 10Full rollout across all buckets
